Perform Hierarchical Clustering¶

Hints: Use linkage() to compute hierarchical clusters using Ward's method, which minimizes variance between clusters. Limit clustering to the first 1000 rows for performance. Use fcluster() to assign cluster labels to data points based on the number of clusters (max_clusters).
In [12]:
print("Performing hierarchical clustering using Ward's method...")
# To Do: in a try/catch block use the Ward's method to perform hierarchical clustering, limit max clusters to 4.

try : 
    wards_np = linkage(normalized_df[:1000].values, method='ward')
    
    clusters = fcluster(wards_np, 4, criterion='maxclust')
    df.loc[:999, 'cluster'] = clusters
except Exception: 
    print("Error Occurred.")
    sys.exit(1)
